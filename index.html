Gestion de documents en ligne de commande Linux

Pour créer un dossier :

mkdir dossier

Pour supprimer un dossier :

rm dossier

Pour déplacer un dossier :

mv dossier

Pour afficher le dossier courant :

pwd

Elevation de droits et gestion des paquets

Pour obtenir les privilèges administrateur :

sudo -s

Pour afficher la liste des paquets :

apt update

Pour mettre à jour les paquets installés :

apt upgrade

Pour rechercher un paquet :

apt search mot_clé

Pour installer un paquet local :

apt install paquet

Git et Github

Git est un outil de versioning permettant une meilleure organisation lorsque plusieurs développeur travaillent ensemble sur une même base de code. Git a été à l'origine créé par Linus Torvalds afin de faciliter l'organisation entre les développeur du kernel Linux. Cet outil a ensuite été mis à disposition de la communauté.

Commandes de base pour Git

Initialisation d'un dépôt :

git init

Ajout d'un dépôt distant :

git remote add origin nom_du_dépôt

Ajout d'un fichier :

git add fichier

Envoi d'un commit :

git commit -m "message"

Envoi du commit vers le dépôt distant :

git push -u origin main

Changement de branche :

git branch nom_branche
